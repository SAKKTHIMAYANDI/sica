ODOO Warehouse Order Split Management

For Providing Easy Way To Manage Multiple Shipment.

Few Words About Order Split:

Order Split Plug-in provides an easy interface for managing the moving lines inside the warehouse. Sometimes due to unavailability of products, providing the delivery of order on the committed time become a tedious issue in such case the warehouse manager can use this plug-in to make the order delivery in multiple batches.

When the customer order quantity more than the available product in the warehouse the seller can split the order and deliver according to the quantity of product available at that time and rest of them later with a new warehouse reference.

Work Flow
Sale Order Creation From Website
Steps For Order Splitting
Select the order you want to split and press (split)
Edit the quantity or Delete the Move Line as per requirements and press(split) button

There can be many cases

Example(s):

  1. If you want to delivered 5 Customizable Desk (White, Steel) Out of 10 later then just edit it from 10 to 5 .
  2. If you want to delivered Only Drawer later then delete other except Drawer.
  3. If you want to delivered All 5 Office Chair Black At Once later then delete other except Office Chair Black.

Splitting Case 1
Deliver 5 Customizable Desk (White, Steel) later
Split Delivery View
Newly Create First Delivery Order

After splitting now you can deliver 5 Customizable Desk (White, Steel) later with a new warehouse reference[WH/OUT/00012] having [WH/OUT/00010] as back order reference.
Newly Create Second Delivery Order

Rest of these product are now ready to deliver with [WH/OUT/00011] warehouse reference having [WH/OUT/00010] as back order reference.
Splitting Case 2
Deliver 5 Customizable Desk (White, Steel) later
Split Delivery View
Newly Create First Delivery Order

After splitting now you can deliver Drawer later with a new warehouse reference[WH/OUT/00013] having [WH/OUT/00011] as back order reference.
Newly Create Second Delivery Order

Rest of these product are now ready to deliver with [WH/OUT/00014] warehouse reference having [WH/OUT/00011] as back order reference.
